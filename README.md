# IoT catProject
# 動機想法
- 家中有飼養貓咪，由於無法與寵物做言語上的溝通，所以觀察身體行為才能了解到貓咪的身體狀況。
- 偶然發現貓咪頻繁進出便盆卻無排泄物，且於便溺時哀號，便懷疑是不是生病了；帶去醫院檢查發現是泌尿道感染、膀胱發炎導致尿液結晶，所以造成貓咪排泄困難。
- 此次實驗希望能掌握貓咪進出便盆次數，提早發現異常，提早治療。

**異常情況**
1.  頻繁進出便盆
2.  便盆無排泄物

# 功能
- 統計貓咪進入便盆次數，當次數異常時，藉由通訊軟體通知飼主，並提供照片觀察情況。
